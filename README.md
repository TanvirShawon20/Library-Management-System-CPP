# 📚 Library Management System (C++ - OOP)

A console-based Library Management System built using **C++ and Object-Oriented Programming (OOP)**.  
This project demonstrates file handling, data structures, and real-world system design concepts.

## 🚀 Features

-->➕ Add new books
-->❌ Delete books
-->📖 View all books
-->🔍 Search books by title
-->📥 Borrow books
-->📤 Return books
-->✏️ Update book information
-->💾 Persistent storage using file handling

## 🧠 Concepts Used

- Object-Oriented Programming (Classes & Objects)
- Encapsulation
- File Handling (`ifstream`, `ofstream`)
- STL (`vector`, `stringstream`)
- Data Serialization & Deserialization


## 🏗️ Project Structure

- `Book` class → handles book data and serialization
- `Library` class → manages all operations (CRUD, search, borrow, return)
- `books.txt` → stores all book records
